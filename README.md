# Integrate with MongoDB - Backend REST API - Practical 6

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman

## Features
- MongoDB Atlas Integration
- Schemas: Product, User, Cart, Order
- CRUD Operations using Mongoose
- Tested using Postman

## Run Locally

1. Clone repo
2. npm install
3. Create .env file
4. node app.js
